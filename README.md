# Tableaux - TP5 - Inverser un tableau

## Consignes

Écrire une fonction `inverser` qui prend en paramètre un tableau de nombres et retourne un nouveau tableau contenant les éléments du tableau d'origine dans l'ordre inverse.

Le tableau d'origine ne doit pas être modifié.

**Signature de la fonction :**

```ts
function inverser(tableau: number[]): number[]
```

## Exemples

- `inverser([1, 2, 3, 4, 5])` doit retourner `[5, 4, 3, 2, 1]`
- `inverser([10, 20])` doit retourner `[20, 10]`
- `inverser([7])` doit retourner `[7]`
- `inverser([])` doit retourner `[]`
